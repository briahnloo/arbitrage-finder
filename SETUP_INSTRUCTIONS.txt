IMPORTANT: Environment Setup

Before running the arbitrage finder, you need to create a .env file:

1. Create a new file called .env in this directory
2. Add the following line to it:

ODDS_API_KEY=your_api_key_here

3. Replace "your_api_key_here" with your actual API key from https://the-odds-api.com/

You can get a free API key by:
- Visiting https://the-odds-api.com/
- Signing up for a free account
- Copying your API key from the dashboard

The free tier includes 500 API calls per month.
